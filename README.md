        <li>Match results</li>

        All data displayed in the app is used solely to:
    </p>
    <ul>
        <li>Show FTC event and match information</li>
        <li>Display team performance metrics</li>
        <li>Enable users to explore competition data</li>
    </ul>
    <h2>4. Data Storage</h2>
    <p>
        The app does not store personal data. Any data retrieved from the FTC API is used temporarily and may be cached locally for performance purposes only.
